Hello this is the testing of git command.
1. git status.
2. staging area: git add filename or git add .
3. commiting area: git commit -m "message" 
4. tracking commit: git log
5. time travelling through commit: git checkout "hash code generated by commit using git log"
6. return original time: git checkout master.
7. creating branch: git branch branch-name
8. checking branch: git branch 
9. switching branch: git checkout branch-name 
10. directly creating branch: git checkout -b swarup/branch-name.
11. merging sub-branch to parent branch: git merge swarup/sum.
12. hiding some file using .gitignore: touch .gitignore
